dependparser Version: 0.0.6 By @yutou

项目的目标是创建一个小程序，自动帮用户分析出一个项目中所require的模块，以帮助用户编写出精确的package.json文件的dependencies和devDependencies属性。

  • index

index: API索引

参数 string source 要分析的项目路径，最后跟上斜线。
参数 function callback 回调方法，其第一个参数如果出错则是一个Error实例，第二个参数是最后获取的依赖的对象。
参数 object _config 配置参数，目前支持blackList（黑名单数组），showError(请求错误是否在控制台显示错误信息）
代码
exports = module.exports  = function(source, callback, _config){
    var dp = new DependParser();
    dp.source = source;
    dp.callback = callback;
    _config&&underscore.extend(dp.config,_config);
    dp.load();
}
